```
Show HN: Agentic Marketplace — Deploy AI workflows with one click

Hi HN,

We built Agentic Marketplace because deploying AI is way too hard.

The problem: You find a great AI tool on GitHub. Then you spend 3 days setting up dependencies, configuring the model API, deploying to the cloud, and monitoring it. Most people give up.

Our solution: A marketplace where AI creators publish pre-built workflows, and businesses deploy them with one click. No code, no infrastructure, no DevOps.

Technical details:
- Backend: FastAPI + Celery + async SQLAlchemy
- Frontend: Next.js 14 + TypeScript + Tailwind
- Infrastructure: PostgreSQL, Redis, Qdrant, MinIO — all containerized
- Auth: JWT with refresh tokens, RBAC with 5 roles
- Multi-tenancy: Tenant isolation at the DB level
- AI execution: OpenRouter API (supports 50+ models)
```
